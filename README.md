# Smart Campus

## Prerequistie
- Python

### Install dependency

- dev

```sh
$ pip install -r requirements/dev.txt
```

### Setup Envrionment Variables
- `SECRET_KEY`

```sh
export SECRET_KEY='Some hard to guess value'
```


## Testing

### Run Test
```sh
$ pytest
```

Note that test should be run in root directory instead of `smart_campus`.
